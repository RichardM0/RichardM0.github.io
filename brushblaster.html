Brush Blaster

No Fuss Tutors

Working at an internship for No Fuss Tutors as a part of the COOP Program at Georges Vanier Secondary School has allowed me to gain professional experience as a game developer. Besides that, I have learned TypeScript and React to create a web-based tower defense game.
Brush Blaster is a tower defense game that allows you to answer questions in between "waves" to get extra brushes, the currency of the game. Besides that, the logic is very similar to other tower defense games. There are waves, a timer that spawns the next wave if you don't beat the current round fast enough, and a multitude of enemies and towers.
The different types of enemies include fast enemies, enemies with a lot of health, balanced, and an enemy that is much stronger than the rest. As for towers, there is a shock tower, laser, cannon, and missile tower.
